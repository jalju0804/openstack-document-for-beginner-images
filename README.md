# Excalidraw 이미지 저장소

이 저장소는 책 제작 과정에서 사용된 Excalidraw 이미지들을 관리하기 위한 공간입니다.

## 폴더 구조

### 📚 `book/`
책을 만들면서 사용한 이미지들을 챕터별로 정리한 폴더입니다.

- 각 챕터는 별도의 폴더로 구분되어 있습니다
- 예: `chapter 1/`, `chapter 2/` 등
- 챕터 폴더 내에는 해당 챕터에서 사용된 Excalidraw 이미지 파일들이 저장됩니다

**예시 구조:**
```
book/
  ├── chapter 1/
  │   └── 오픈스택이란?.md
  └── chapter 2/
      └── ...
```

### 📝 `docs/`
책을 만들기 위해 스터디에서 사용된 이미지들을 모아둔 폴더입니다.

- 각 파일의 이름은 노션(Notion)의 제목과 1대1로 매핑됩니다
- 스터디 자료로 사용된 Excalidraw 이미지들을 저장합니다

**예시:**
```
docs/
  ├── proxmox 네트워크 설정.md  (← 노션 제목: "proxmox 네트워크 설정")
  └── ...
```

## 네이밍 제안

현재 폴더 구조가 명확하고 직관적이지만, 다음과 같은 대안도 고려해볼 수 있습니다:

### `book/` 폴더 대안
- `book-assets/` - 책 관련 에셋임을 명확히 표현
- `book-images/` - 이미지 파일임을 명시
- `chapter-assets/` - 챕터별 에셋임을 강조

### `docs/` 폴더 대안
- `study-assets/` - 스터디 자료임을 명확히 표현
- `study-images/` - 스터디 이미지임을 명시
- `notion-assets/` - 노션과의 매핑 관계를 강조

현재 네이밍(`book/`, `docs/`)도 충분히 명확하고 간결하므로, 변경하지 않아도 무방합니다.

## 사용 방법

1. **책 이미지 추가**: 해당 챕터 폴더에 Excalidraw 파일을 저장하세요
2. **스터디 이미지 추가**: `docs/` 폴더에 노션 제목과 동일한 이름으로 파일을 저장하세요
